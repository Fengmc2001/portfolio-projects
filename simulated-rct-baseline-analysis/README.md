# Simulated Clinical Trial Data Analysis

This directory contains the R scripts and raw dataset (CSV) used for analyzing a simulated randomized controlled trial (RCT). The project involves baseline balance checks, pairwise scatter plot generation, paired and independent t-tests for pre/post-intervention blood pressure changes, and power analysis for sample size calculation.

## Contents
- **R Scripts**: Contains the logic for descriptive statistics and statistical inference testing using R (`cards[7].R`, `kadai67.R`).
- **Data**: The `.csv` files are simulated datasets containing patient information (Age, Sex, Systolic Blood Pressure) before and after the intervention.
