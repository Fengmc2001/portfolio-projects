# プロジェクト：項目反応理論（IRT）に基づく動的 e-Testing システム

## 1. 概要

本プロジェクトは、項目反応理論（Item Response Theory: IRT）を用いて、受験者の解答パターンから潜在的な能力値を推定するブラウザベースの e-Testing プロトタイプである。

従来の単純な正答率による評価では、問題の難易度や識別力を十分に考慮できない。本プロジェクトでは、各問題に識別力パラメータ `a` と困難度パラメータ `b` を与え、2パラメータロジスティックモデルに基づいて受験者の能力値 `theta` を推定する。

## 2. 理論的背景

本システムでは、2パラメータロジスティックモデルを用いる。

```math
P_i(\theta) = \frac{1}{1 + \exp(-D a_i(\theta - b_i))}
```

ここで、`theta` は受験者の潜在能力、`a_i` は項目の識別力、`b_i` は項目の困難度、`D` はスケーリング定数である。

受験者の解答履歴に対して、各能力値候補における尤度を計算し、事前分布と組み合わせることで、能力値の事後分布を求める。最終的には、事後確率が最大となる能力値を推定値として採用する。

## 3. 実装内容

本プロジェクトでは、HTML と JavaScript を用いて、以下の機能を実装した。

- 問題バンクの読み込み
- 2PL モデルによる正答確率の計算
- Bayes 的な能力値推定
- 未回答・スキップ選択肢の処理
- 回答時間の記録
- 自信度アンケートの取得
- 解答履歴に基づく戻る機能
- テスト終了後の結果表示

## 4. 工夫した点

### 4.1 未回答処理

選択肢に「わからない」を設け、適当な解答によって能力値が不当に低く推定されることを防いでいる。未回答は `-1` として記録し、能力値推定時には除外する。

### 4.2 状態保存と復元

各設問の解答前に、問題番号、能力値、解答履歴、出題履歴、回答時間、自信度などを `exam.history` に保存している。戻るボタンを押すと、この履歴スタックから直前の状態を復元し、画面を再描画する。

### 4.3 回答時間と自信度の記録

単に正誤だけを見るのではなく、回答時間や主観的な自信度も記録することで、今後の分析や拡張に利用できる構造にしている。

## 5. ファイル構成

```text
src/
├── index.html
├── 1_functions.js
├── 2_itemBank.json
└── 3_functions.js
```

- `index.html`: テスト画面の HTML
- `1_functions.js`: IRT の確率計算、Bayes 推定、シミュレーション関数
- `2_itemBank.json`: 問題、選択肢、正答、項目パラメータ
- `3_functions.js`: テスト進行、状態管理、結果表示の処理
